概要

RX-01 Playerは、語学学習と高度な動画分析に最適化された次世代動画プレイヤーです。わずか10MBの軽量設計ながら、強力な機能を備えています。

主な特徴

  • 最大3つまでの字幕を同時表示
  • 字幕から辞書・発音サイトへ瞬間検索
  • バレットタイムリープ機能で超スロー再生中の巻き戻しも快適
  • ホイール操作でマウスの位置を最大3倍ズームイン可能
  • Picture-in-Picture対応
  • 10MBの軽量設計で爆速起動
  • 豊富なショートカットキー（右クリックでキーバインドの変更可能）

バージョン比較

RX-01 Playerは、利用シーンに応じて4つのバージョンを提供しています。

Windows版

推奨

デスクトップアプリケーションとして動作するネイティブ版です。

特徴

  • OSネイティブ動作で最高のパフォーマンス
  • Windows 10/11に対応
  • わずか10MBのインストールサイズ
  • 全機能を完全サポート

Mac版

推奨

macOS向けのネイティブアプリケーション版です。

特徴

  • macOS最適化で快適動作
  • macOS 10.15 Catalina以降に対応
  • Apple Siliconネイティブサポート
  • 約10MBの超軽量設計
  • 全機能を完全サポート

VisiOS版

Web

Chrome拡張機能「VisiOS」上で動作するブラウザ版です。

特徴

  • １秒インストールでブラウザから利用
  • タブやブックマークを視覚的に管理
  • 他のWebアプリと連携して動作
  • Chrome/Edge/Braveなど対応
  • クロスプラットフォーム対応
  • VisiOS拡張機能が必要

Nyan-8版

Social

コミュニケーション機能を統合したブラウザ版です。

特徴

  • 完全インストール不要でブラウザから即利用
  • 他のユーザーとテキストメッセージ
  • ビデオチャット機能搭載
  • ボイスコール対応
  • 動画を見ながら友達と会話

音楽形式

💡 動画ファイルの読み込み方法

  • ドラッグ&ドロップでファイルを追加
  • ファイル選択ダイアログから選択
  • URLを直接入力して再生 (動画再生中のCtrl+Vや右クリックメニューからの貼り付けにも対応)
  • .srtファイルをドロップして字幕を追加（最大3つまで）

主な機能

トリプル字幕表示

最大3つまでの字幕を同時に表示できます。.srtファイルをドロップするだけで簡単に読み込めます。

使用例：英語学習で英語字幕、日本語字幕、発音記号を同時表示

辞書連携機能

字幕内の単語や文章を、登録した辞書サイトや発音サイト、文法サイトから瞬時に検索できます。

  • 100種類以上のサイトをプリセット
  • カスタムサイトの追加が可能
  • 単語選択して右クリックで検索
  • 文全体や一部分だけを選択可能

バレットタイムリープ

聞き取れなかった部分をスロー再生で繰り返し確認できる機能です。

  • 0.1x～4.0xで速度調整（YouTube動画は0.25～2.0x）
  • 再生速度に応じた0.3～3秒の巻き戻しに対応してるので一瞬の動作の確認が簡単
  • 語学学習に最適
  • ダンスやマジックの動作確認にも有効

YouTubeのCCをダウンロード

YouTube再生中、一時停止したら右上からいつでも字幕をダウンロード可能

  • 再生中、字幕が必要になったら右上の.srtダウンロードボタンをクリック
  • 欲しい言語の.srtをダウンロードして、プレイヤーにドロップするだけで字幕表示

Picture-in-Picture

動画を小窓で表示し、他の作業をしながら視聴できます。

  • デスクトップ版・ブラウザ版で対応
  • サイズ・位置を自由に調整

シークバープレビュー

シークバーにマウスをホバーすると、その位置の映像をプレビュー表示します。

ショートカットキー

豊富なショートカットキーで快適に操作できます。ショートカットキーのキーバインディングは画面を右クリックして「設定」を選ぶといつでも変更可能です

再生制御

Space または S 再生/一時停止
← または A 3秒戻る（再生速度に応じて可変）
→ または D 3秒進む（再生速度に応じて可変）

音量・速度

↑ / ↓ 音量を上げる/下げる
M ミュート切替
[ / ] 再生速度を下げる/上げる

表示

F フルスクリーン切替
C .srtの字幕表示切替
P Picture-in-Picture

その他
