# RX-01 Player - 究極完全体の言語学習メディアプレイヤー

## 概要
RX-01 Playerについては公式サイトをぜひ見てください。
https://rx-01.vercel.app/

## 機能
- 📹 ローカルビデオファイルの再生（MP4, WebM, OGG, MKV, MOV対応）
- 🎵 オーディオファイルの再生（MP3, WAV, FLAC, AAC対応）
- 🌐 URLからの動画/音楽読み込み
- 🎯 ドラッグ＆ドロップによるファイル読み込み
- ⌨️ 豊富なキーボードショートカット
- 🎮 再生速度調整（0.1x～4.0x）
- 📊 プログレスバーとタイムスタンプ表示
- 🔊 音量調整とミュート機能
- 🖥️ フルスクリーン対応

## キーボードショートカット
- **Space / S** - 再生/一時停止
- **A / ←** - 3秒戻る
- **D / →** - 3秒進む
- **↑ / ↓** - 音量調整 (±10%)
- **M** - ミュートトグル
- **F** - フルスクリーン切替
- **P** - 速度を1.0xにトグル
- **[ / ]** - 速度調整 (±0.1x)
- **Ctrl + V** - URLから読み込み（URL入力欄にフォーカス）


## 使い方

### ファイルの読み込み方法
1. **ファイルを開くボタン** - ダイアログからファイルを選択
2. **ドラッグ＆ドロップ** - ファイルをウィンドウにドロップ
3. **URL入力** - 動画/音楽のURLをコピペするだけでロードされる

### 対応フォーマット
- **ビデオ**: MP4, WebM, OGG, MKV, MOV, AVI
- **オーディオ**: MP3, WAV, FLAC, AAC, OGG
